# Customer Retention & Revenue Analysis (SQL)

## Tools Used

* SQL (BigQuery)
* Google Sheets

## Project Overview

This project analyzes customer behavior, revenue trends, and product performance using SQL. The goal is to understand retention patterns, customer value, and key revenue drivers.

## Key Analyses

* Monthly Revenue Trend
* Customer Retention (Repeat vs One-Time)
* Top Customers by Spending
* Customer Lifetime Value (CLV)
* Revenue by Product

## Key Insights

* A portion of customers are repeat buyers, contributing significantly to revenue
* Revenue is concentrated among top-performing customers and products
* Monthly trends show fluctuations indicating seasonal behavior

## Files Included

* SQL queries used for analysis
* Query outputs (CSV files)
* Cleaned dataset structure

## Skills Demonstrated

* SQL JOINs
* Aggregations (SUM, COUNT, AVG)
* Window functions (RANK)
* Subqueries and CASE statements
* Business data analysis
